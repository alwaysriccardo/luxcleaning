# Lux Cleaning & Hauswartung

Professionelle Reinigungsdienstleistungen in der Schweiz mit deutscher Pünktlichkeit und höchster Präzision.

## Features

- Modern, responsive design with smooth animations
- Service showcase with detailed descriptions
- Customer reviews section
- Contact form and quote request
- German and Swiss market focus

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Preview production build:
   ```bash
   npm run preview
   ```

## Project Structure

- `index.tsx` - Main React application component
- `index.html` - HTML entry point with styles
- `constants.tsx` - Service and review data (optional)
- `types.ts` - TypeScript type definitions
- `vite.config.ts` - Vite configuration
